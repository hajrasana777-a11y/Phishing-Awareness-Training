#Phishing Awareness Training
## Project overview
This project is a phishing awareness website created as a part of the CodeAlpha Cyber Security Internship.

## Features
- Explain phishing attacks
- Identify phishing emails
- Detect fake websites
- Social engineering awareness 
- Best security practices
- Real-world phishing example
- interactive quiz
- Responsive design

## Technologies used
- HTML
- CSS
-JavaScript

## How to run
1. Download the project.
2. Open the project folder in VS Code.
3. Open index.html using Live Server.
